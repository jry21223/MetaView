# Math Theorem Walkthrough

## Use it for

- Calculus, limits, derivatives, integrals
- Geometry, analytic geometry, linear algebra
- Any math explanation that depends on symbolic derivation and diagram consistency

## Workflow

1. Define the axes, geometric objects, and symbolic notation before transforming anything.
2. Use MathTex-style symbolic continuity for every derivation step.
3. For calculus, explicitly track the moving variable, tangent, area, or accumulation region.
4. Close with the formal result plus the intuitive interpretation.

## Guardrails

- Never jump over algebraic steps that change meaning.
- Keep the rendered object numerically consistent with the stated formula.
- Avoid mixing multiple coordinate systems in one unexplained frame.
