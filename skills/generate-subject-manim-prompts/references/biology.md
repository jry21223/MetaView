# Biology Prompt Guidance

Use this file for cells, organs, pathways, genetics, ecology, anatomy, and staged biological processes.

## Priorities

- Make the scale explicit: molecule, organelle, cell, tissue, organism, or ecosystem.
- Keep stage order clear for cycles, pathways, and developmental processes.
- Use labels that distinguish structure from function.
- Prefer a guided progression over a dense one-frame infographic.

## Prompt Additions

- Ask for a hierarchy map listing the biological level and the main entities.
- Ask for a process timeline or pathway order before code.
- Require arrows to have one meaning per scene: flow, signaling, transport, inheritance, or change over time.
- Ask for concise callouts explaining the role of each stage or structure.
- Require a final recap of the biological mechanism in plain language.

## Failure Patterns To Prevent

- Switching scale without warning
- Overusing color without a legend
- Arrows with mixed meanings in one frame
- Process steps shown out of order
